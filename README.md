## How to Start

```
// Next.js 실행
git clone https://github.com/dnd-side-project/dnd-8th-7-frontend
yarn install
yarn dev

// React Native 실행 (ios)
git clone https://github.com/dnd-side-project/dnd-8th-7-frontend-app
yarn install
cd ios
pod install
cd .. && yarn ios
```
